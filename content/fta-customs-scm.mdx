## 관세를 모르면 원가 계산이 틀린다

해외에서 원단을 1억 원어치 수입한다고 하자. 관세율이 8%면 관세 800만 원, 부가세 1,080만 원. 도입 원가는 **1억 1,880만 원**이 된다.

그런데 FTA를 적용하면 관세가 0~3%로 낮아질 수 있다. 관세 0%면 부가세만 1,000만 원. 도입 원가 **1억 1,000만 원**. 차이 880만 원.

연간 수입 금액이 수십억이면, FTA 활용 여부에 따라 **수억 원의 비용 차이**가 난다. SCM 담당자가 관세와 FTA를 알아야 하는 이유다.

## 관세의 기본 구조

### HS Code (Harmonized System Code)

모든 국제 무역 품목에는 HS Code가 부여된다. 전 세계 공통의 분류 체계다.

- 6자리까지: 국제 공통
- 7~10자리: 각 나라별 세분류

**예시**: 가구업체에서 수입하는 원단
- 5407: 합성섬유 직물
- 5407.61: 폴리에스터 단섬유 직물 (85% 이상)
- 5407.61.0000: 한국 관세율 분류

HS Code가 잘못되면 관세율이 달라지고, 관세 추징이 발생할 수 있다. **HS Code 분류는 관세사에게 확인**하는 것이 안전하다.

### 관세율의 종류

| 종류 | 설명 |
|------|------|
| 기본 관세율 | 관세법에 정해진 기본 세율 |
| WTO 양허세율 | WTO 회원국에 적용되는 세율 (기본보다 낮은 경우가 많음) |
| FTA 특혜세율 | FTA 체결국에 적용되는 우대 세율 (0~낮은 세율) |
| 할당관세 | 특정 기간·수량에 한해 낮은 세율 적용 |
| 조정관세 | 산업 보호 등을 위한 일시적 세율 조정 |

실제 적용 세율 = min(기본, WTO, FTA, 할당, 조정) 중 가장 낮은 것을 선택.

### 수입 비용의 구조

```
CIF 가격 (물건값 + 보험 + 운임)
  + 관세 = CIF × 관세율
  + 부가가치세 = (CIF + 관세) × 10%
  + 통관 수수료, 검역비, 보관료 등
  = 총 도입 원가
```

## FTA란 무엇인가

FTA(Free Trade Agreement)는 국가 간 관세를 줄이거나 없애는 협정이다.

### 한국의 주요 FTA 현황

한국은 60개 이상의 국가와 FTA를 체결했다. SCM에서 중요한 것들:

| FTA | 상대국 | 발효 | 주요 혜택 |
|-----|--------|------|----------|
| 한-중 FTA | 중국 | 2015 | 많은 품목 관세 인하·철폐 (단계적) |
| 한-베 FTA (VKFTA) | 베트남 | 2015 | 섬유·원단 관세 인하 |
| 한-EU FTA | EU 27개국 | 2011 | 대부분 공산품 관세 철폐 |
| 한-미 FTA | 미국 | 2012 | 공산품 대부분 관세 철폐 |
| RCEP | 아시아 15개국 | 2022 | 역내 원산지 누적 가능 |
| 한-ASEAN FTA | 아세안 10개국 | 2007 | 인도네시아, 말레이시아 등 |

### FTA 특혜 관세를 받으려면

FTA 특혜 관세는 자동으로 적용되지 않는다. **원산지 증명을 해야** 받을 수 있다.

## 원산지의 이해

### "원산지"가 왜 복잡한가

"중국에서 만들었으니 중국산"이 단순할 것 같지만, 실제로는 복잡하다.

**예시**: 인도네시아산 원목을 중국에서 가공해서 합판을 만들었다. 이것은 중국산인가, 인도네시아산인가?

원산지를 결정하는 기준이 FTA마다 다르고, 품목마다 다르다.

### 원산지 결정 기준

#### 1. 완전 생산 기준 (Wholly Obtained)
해당 국가에서 완전히 생산된 물품. 농산물, 광물 등에 적용.

#### 2. 세번 변경 기준 (Change in Tariff Classification)
원재료의 HS Code와 완제품의 HS Code가 다르면 해당 국가에서 "실질적 변형"이 이루어진 것으로 인정.

**예시**: 원사(HS 5402) → 직물(HS 5407). HS Code가 변경되었으므로 직물의 원산지는 제직(weaving)이 이루어진 국가.

#### 3. 부가가치 기준 (Regional Value Content)
해당 국가에서 일정 비율 이상의 부가가치가 창출되면 원산지 인정.

**예시**: RVC 40% 이상. 제품 가격 100달러 중 40달러 이상의 가치가 해당 국가에서 추가되어야 원산지 인정.

#### 4. 특정 공정 기준 (Specific Process Rule)
특정 공정이 해당 국가에서 이루어지면 원산지 인정. 섬유 분야에서 많이 사용.

### 원산지 증명서

FTA 특혜세율을 받으려면 원산지 증명서가 필요하다.

| 유형 | 발급자 | 적용 FTA |
|------|--------|---------|
| 기관 발급 | 세관, 상공회의소 | 한-중, 한-ASEAN |
| 자율 발급 | 수출자/생산자 | 한-EU, 한-미 |
| 인증 수출자 | 관세청 인증을 받은 수출자 | 한-EU |

## FTA 활용 실무 프로세스

### 수입 시 FTA 활용 (우리가 해외에서 사올 때)

```
1. 수입 품목의 HS Code 확인
2. 해당 국가와의 FTA 확인 (특혜세율 존재 여부)
3. FTA 특혜세율 확인 (관세청 FTA 포털)
4. 공급업체에 원산지 증명서 요청
5. 수입 통관 시 FTA 특혜세율 적용 신청
6. 원산지 증명서 5년 보관
```

### 수출 시 FTA 활용 (우리 제품을 해외로 팔 때)

```
1. 수출 품목의 HS Code 확인
2. 수출국과의 FTA 확인
3. 원산지 결정 기준 충족 여부 확인
4. 원산지 소명서 작성 (원재료, 공정 기록)
5. 원산지 증명서 발급 또는 자율 증명
6. 관련 서류 5년 보관
```

### 가구업체에서 자주 하는 실수

1. **원산지 증명서를 안 받는다**: 공급업체에 요청하지 않아서 FTA 혜택을 못 받음
2. **HS Code가 틀리다**: 관세율이 달라져서 추징 발생
3. **서류를 보관하지 않는다**: 관세청 사후 검증 시 서류 미비로 특혜 취소
4. **원산지 결정 기준을 모른다**: 재료의 원산지와 가공국이 다를 때 판단을 못 함

## 가구업체 관세 실무 사례

### 사례 1: 중국산 원단 수입

품목: 폴리에스터 원단 (HS 5407.61)
기본 관세율: 10%
한-중 FTA 특혜세율: 5% (2026년 기준, 단계적 인하)
연간 수입액: 5억 원

| | FTA 미적용 | FTA 적용 |
|---|-----------|---------|
| 관세 | 5,000만 원 | 2,500만 원 |
| 부가세 | 5,500만 원 | 5,250만 원 |
| **절감액** | - | **2,750만 원** |

원산지 증명서 한 장으로 2,750만 원을 아낄 수 있다.

### 사례 2: 한국산 가구 미국 수출

한-미 FTA로 대부분의 가구류 관세 0%.
하지만 가구에 사용된 원단이 중국산이라면? 원산지 결정 기준에 따라 "한국산"으로 인정되는지 확인 필요.

한국에서 중국산 원단을 사용해 소파를 제조 → 세번 변경 기준 (원단 HS → 가구 HS)으로 한국산 인정 가능.

## SCM 담당자를 위한 체크리스트

### 수입 시

- [ ] 수입 품목의 HS Code를 관세사와 확인했는가
- [ ] FTA 특혜세율이 있는지 확인했는가 (관세청 FTA 포털)
- [ ] 공급업체에 원산지 증명서를 요청했는가
- [ ] 원산지 증명서의 유효기간과 기재 내용이 정확한가
- [ ] 통관 시 FTA 적용을 관세사에게 요청했는가

### 수출 시

- [ ] 수출 품목이 FTA 특혜 대상인지 확인했는가
- [ ] 원산지 결정 기준을 충족하는지 확인했는가
- [ ] 원재료 원산지 소명 자료를 보관하고 있는가
- [ ] 원산지 증명서를 발급/자율 증명했는가

---

## 정리

관세와 FTA는 SCM 비용에 직접 영향을 미친다.

- **HS Code**를 정확히 분류해야 정확한 관세율을 적용받는다
- **FTA 특혜세율**은 자동 적용이 아니다 — 원산지 증명이 필요하다
- **원산지 결정 기준**은 FTA마다, 품목마다 다르다
- **서류 보관 5년**은 필수다 (사후 검증 대비)

"관세는 관세사가 알아서 해주겠지"가 아니라, SCM 담당자가 기본을 이해하고 있어야 TCO를 정확히 계산하고, 비용을 줄일 수 있다.
