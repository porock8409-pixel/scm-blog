## "FOB 선전, CIF 부산" — 이게 뭔 뜻인가

해외 공급업체에게 견적을 요청하면 이런 답이 온다.

"Unit price: $5.20 FOB Shenzhen"
"Unit price: $5.80 CIF Busan"

같은 물건인데 가격이 다르다. $5.20이 더 싼 것 같지만, 실제로는 그렇지 않을 수 있다. **FOB와 CIF에 포함된 비용의 범위가 다르기** 때문이다.

인코텀즈(Incoterms)를 모르면 해외 거래에서 **비용과 리스크가 누구 책임인지** 판단할 수 없다.

---

## 인코텀즈란

인코텀즈(Incoterms, International Commercial Terms)는 ICC(국제상업회의소)가 제정한 **국제 무역 거래 조건의 표준**이다. 현재 버전은 Incoterms 2020.

인코텀즈가 정하는 것:
1. **비용 분담**: 운송비, 보험료를 누가 부담하는가
2. **리스크 이전**: 물건이 손상/분실되면 누구 책임인가
3. **의무**: 수출 통관, 수입 통관을 누가 하는가

인코텀즈가 정하지 않는 것:
- 물건의 소유권 이전 시점
- 대금 결제 조건 (L/C, T/T 등)
- 분쟁 해결 방법

---

## 핵심 인코텀즈 4가지

인코텀즈 2020에는 11가지 조건이 있지만, 실무에서 자주 쓰이는 것은 4가지다.

### EXW (Ex Works, 공장 인도)

**의미**: 판매자 공장에서 물건을 인도. 그 이후의 모든 비용과 리스크는 구매자.

```
[판매자 공장] ← 판매자 책임 여기까지
→ 국내 운송 → 수출 통관 → 국제 운송 → 수입 통관 → 국내 운송 → [구매자]
              ← 구매자 책임: 전부 →
```

**장점**: 물건 가격이 가장 싸다 (운송비, 보험 미포함)
**단점**: 구매자가 모든 물류를 직접 관리해야 한다. 수출 통관까지 구매자 책임.
**적합한 경우**: 현지에 물류 네트워크가 있거나, 직접 물류를 통제하고 싶을 때

### FOB (Free On Board, 본선 인도)

**의미**: 판매자가 지정 선적항에서 배에 물건을 실을 때까지 책임. 배에 실린 이후는 구매자 책임.

```
[판매자 공장] → 국내 운송 → 수출 통관 → [선적항 - 배에 적재]
← 판매자 책임 ──────────────────────── →
                                        → 국제 운송 → 수입 통관 → 국내 운송 → [구매자]
                                        ← 구매자 책임 ────────────────────── →
```

**장점**: 가장 많이 사용되는 조건. 책임 분담이 명확.
**단점**: 구매자가 해상 운송 이후를 관리해야 한다.
**적합한 경우**: 한국에서 포워더를 직접 지정해서 운임을 통제하고 싶을 때

**가구업체 실무**: "FOB 선전(Shenzhen)"이면, 공급업체가 선전 항구까지 운송하고 수출 통관까지 해준다. 선적 이후 해운 운임, 보험, 수입 통관은 우리가 한다.

### CIF (Cost, Insurance and Freight, 운임보험료 포함)

**의미**: 판매자가 지정 도착항까지의 운임과 보험료를 부담. 하지만 리스크는 선적 시점에 이전.

```
[판매자 공장] → 국내 운송 → 수출 통관 → [선적항] → 국제 운송 → [도착항]
← 판매자 비용 부담 ────────────────────────────────────── →
← 판매자 리스크 ─────────────────── → (선적 시점까지)
                                                        → 수입 통관 → 국내 운송 → [구매자]
                                                        ← 구매자 비용·리스크 ────── →
```

**주의**: CIF에서 비용은 도착항까지 포함이지만, **리스크는 선적 시점에 구매자에게** 넘어간다. 항해 중 물건이 손상되면 구매자가 보험으로 처리해야 한다.

**장점**: 도착항까지 비용이 포함되어 비교가 쉽다. 관세 계산 기준이 CIF.
**단점**: 운임과 보험을 판매자가 잡기 때문에 비용 통제가 어려울 수 있다.
**적합한 경우**: 물류 경험이 적고, 판매자에게 운송까지 맡기고 싶을 때

### DDP (Delivered Duty Paid, 관세 지급 인도)

**의미**: 판매자가 구매자 지정 장소까지 모든 비용과 리스크를 부담. 수입 관세까지 포함.

```
[판매자 공장] → 국내 운송 → 수출 통관 → 국제 운송 → 수입 통관 → 국내 운송 → [구매자]
← 판매자 책임: 전부 ─────────────────────────────────────────── →
```

**장점**: 구매자 입장에서 가장 편하다. 문 앞까지 배달.
**단점**: 물건 가격이 가장 비싸다. 물류에 대한 통제력이 없다.
**적합한 경우**: 소량 구매, 이커머스 직구매, 물류를 전혀 관리하고 싶지 않을 때

## 가구업체 실무에서의 선택

### FOB가 가장 많이 쓰이는 이유

한국 가구업체의 수입 거래에서 **FOB가 가장 일반적**이다. 이유:

1. **운임 통제**: 한국에서 포워더를 지정해서 운임을 직접 협상
2. **보험 통제**: 필요한 수준의 보험을 직접 가입
3. **물류 가시성**: 선적 이후 추적을 직접 관리
4. **비용 투명성**: 물건 가격과 물류비가 분리되어 비교 용이

### CIF를 쓰는 경우

- 처음 거래하는 국가/공급업체 (물류 경험 부족)
- 공급업체가 물류까지 잘 하는 경우
- 관세 계산이 필요할 때 (관세 = CIF × 세율)

### 주의사항

**"FOB가 CIF보다 싸다"고 단정하지 마라.** FOB 가격에 직접 잡은 운임을 더한 것이 CIF보다 비쌀 수 있다. 특히 소량 화물일수록 자체 운임이 비쌀 수 있다.

## 국제 운송의 기본

### 해상 운송 (Ocean Freight)

가구업체 수입의 대부분.

**컨테이너 종류**:
- 20ft (TEU): 내부 약 33CBM. 소량 화물
- 40ft: 내부 약 67CBM. 가구 부자재에 적합
- 40ft HC (High Cube): 내부 약 76CBM. 높이 1ft 추가. 부피가 큰 가구에 유용

**운임 구조**:
- 해상 운임 (Ocean Freight): 컨테이너당 또는 CBM당
- BAF (Bunker Adjustment Factor): 유류 할증
- THC (Terminal Handling Charge): 항구 하역비
- CFS Charge: 혼적(LCL) 화물 처리비

**리드타임**:
| 출발 | 도착 | 소요 기간 |
|------|------|----------|
| 중국 (상하이/선전) | 부산 | 3~5일 |
| 베트남 (호치민) | 부산 | 7~10일 |
| 인도 (뭄바이) | 부산 | 14~21일 |
| 유럽 (함부르크) | 부산 | 28~35일 |

### 항공 운송 (Air Freight)

해상 대비 5~10배 비싸지만, 긴급할 때 사용.

**적합한 경우**:
- 긴급 자재 (생산 라인 대기 중)
- 고가·경량 품목 (고급 하드웨어)
- 샘플 운송

**리드타임**: 대부분 2~5일

### FCL vs LCL

| | FCL (Full Container Load) | LCL (Less than Container Load) |
|---|--------------------------|-------------------------------|
| 의미 | 컨테이너 전체 사용 | 다른 화주와 컨테이너 공유 |
| 비용 | 컨테이너당 고정 | CBM당 과금 |
| 적합 | 물량이 많을 때 | 소량 화물 |
| 리드타임 | 직항 가능 | CFS 경유, 2~5일 추가 |

가구업체 팁: 20ft 컨테이너의 60% 이상을 채울 수 있으면 FCL이 LCL보다 경제적인 경우가 많다.

## 결제 조건

### T/T (Telegraphic Transfer, 전신환 송금)

가장 간단한 결제 방식. 은행 송금.

- **선급 T/T**: 선적 전 대금 지급. 판매자에게 유리
- **후급 T/T**: 선적 후 (보통 B/L 수령 후 30~60일) 대금 지급. 구매자에게 유리
- **분할 T/T**: 계약 시 30%, 선적 시 70% 등으로 분할

실무에서는 **T/T 30% deposit + 70% upon B/L copy** (발주 시 30%, 선적서류 수령 시 70%)가 흔하다.

### L/C (Letter of Credit, 신용장)

은행이 대금 지급을 보증하는 방식. 대규모 거래, 신규 거래에서 안전.

- 은행 수수료가 발생 (개설 수수료 + 확인 수수료)
- 서류 조건이 엄격 (서류 불일치 시 대금 지급 거절)
- 규모가 큰 첫 거래에서 사용 추천

## 정리

국제 물류를 이해하려면:

1. **인코텀즈**로 비용과 리스크 분담을 이해하라 — FOB가 실무 기본
2. **해상 운송**의 구조를 알아라 — 컨테이너, 운임, 리드타임
3. **결제 조건**을 협상하라 — T/T 조건에 따라 현금 흐름이 달라진다
4. **총비용**으로 비교하라 — FOB 가격 + 운임 + 보험 + 통관 = 실제 도입 원가

"FOB 5달러"와 "CIF 6달러" 중 어떤 게 싼지 답하려면, FOB에 직접 운임과 보험을 더해서 비교해야 한다. 그 계산을 할 수 있으면 국제 물류의 기본은 된 것이다.
